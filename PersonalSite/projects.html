A Cappella Website

This website was designed to help the male a cappella group at ASU keep track of their events, meetings, and repertoires. The site used a SQL database with PHP to access and make changes using forms and tables. We used embedded html, and css, and made it colorful, and ASU themed to attract use.

Hill Cipher Matrix Cryptography

This program was developed in Java using Eclipse. It takes as a simple input of a matrix of numbers representing characters and multiplies it by a matrix “key” and returns the encrypted numbers to be converted to text. I also performed decryption by the reverse process of taking the inverse of the key matrix.

x86 Processor Simulator

This program was developed in C with one other team member. We used a simplified x86 instruction set and a 5 stage pipeline machine to increase throughput by allowing multiple instructions to be handled at a time. The 5 stages were fetch, decode, execute, access, and write.
